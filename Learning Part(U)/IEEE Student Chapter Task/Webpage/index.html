What is DevOps?

Defination:

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops).
It aims to shorten the systems development life cycle and provide continuous delivery with high software quality.
DevOps is complementary with Agile software development; several DevOps aspects came from the Agile methodology.

Purpose of DevOps

The DevOps Life Cycle

  • Planning (or ideation)
  • Development.
  • Integration (or build, or continuous Integration and continuous delivery (CI/CD).
  • Deployment (usually called continuous deployment)
  • Operations.
  • Learning (sometimes called continuous feedback)
